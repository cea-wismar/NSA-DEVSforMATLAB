# Examples of Basic Paper
This directory contains all examples from the article [Implementing Standard
Examples with NSA-DEVS](https://www.researchgate.net/publication/366634364_Implementing_Standard_Examples_with_NSA-DEVS)

Since the NSA-DEVS modelbase is enhanced continuously, the paper models might
not work eventually and would have to be adapted. Therefore, they use a fixed
library StdLib that contains copies of the necessary components.


## A Comparator-Switch Model

Models:

- compswitch: standard model described in the article
- compswitchA: has additional gain before the comparator
- compswitchB: uses switch without empty entities
