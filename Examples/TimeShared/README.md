# Example TimeShared

This directory contains all examples from the article [Using Component-based
Discrete-event Modeling with NSA-DEVS – an
Invitation](https://www.researchgate.net/publication/401474702_Using_Component-based_Discrete-event_Modeling_with_NSA-DEVS_-_an_Invitation)

Since the NSA-DEVS modelbase is enhanced continuously, the paper models might
not work eventually and would have to be adapted. Therefore, they use a fixed
library StdLib that contains copies of the necessary components.
