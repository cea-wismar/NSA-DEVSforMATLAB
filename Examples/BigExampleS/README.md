# Model of "Real-World Application" Paper, Small Version

This directory contains a downsized version of the production line model from
the article [Modeling and Simulation of a Real-world Application using
NSA-DEVS](https://www.researchgate.net/publication/376651034_Modeling_and_Simulation_of_a_Real-world_Application_using_NSA-DEVS).

It uses the standard NSA-DEVS modelbase and might have to be adapted after
changes of the modelbase. Many components have smaller parameter values to
reduce the runtime of tests.
